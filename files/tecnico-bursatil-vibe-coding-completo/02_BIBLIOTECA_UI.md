PARTE 5: RESPUESTA COMPLETA A ROSARIO (TEMPLATE)
═════════════════════════════════════════════════════════════════════════════════

COPIA Y PEGÁ ESTO EN ROSARIO, REEMPLAZANDO [CORCHETES]:

"Rosario, necesito componentes React con:

TEMA GENERAL: [Dark Profesional / Light Minimalista / Glassmorphism]

PALETA:
- BUY: [color hex]
- HOLD: [color hex]
- SELL: [color hex]
- Primary: [color hex]
- Background: [color hex]

TIPOGRAFÍA: [Fintech Clásica / Moderno Bold / Luxury]

COMPONENTES:
1. Semáforo: [Simple Badge / Button Prominent / Animated Pill]
2. Tabla: [Tabla Clásica / Cards Grid / Tabla Compacta]
3. Detail View: [Side Panel / Modal Centered / Full Page]
4. Portfolio: [Resumen + Lista / Pie Chart / Cards Grandes]

REQUISITOS ADICIONALES:
- Responsive (375px mobile - 1440px desktop)
- TailwindCSS
- Interaction: hover, click, animaciones suaves

Devuelve 4 componentes React (uno para cada sección)"

═════════════════════════════════════════════════════════════════════════════════

Después que Rosario devuelve los 4 componentes (bloques negros),
vos los pasás a Claude Code con:

"Aquí 4 componentes React. Intégralos en:
- src/components/StockGrid.tsx
- src/components/DetailView.tsx
- src/components/Portfolio.tsx
- src/pages/index.tsx (importa todos)"
